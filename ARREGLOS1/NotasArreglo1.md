Las direcciones de datos se leen como enteros %d o %i
Por cada brinco de datos, la direccion decrementa 4 bits
TIPO FLOAT (4 BITS), ETIQUETA TEMP DIR, CARACTER [24]; BITS DE TIPO X CARACTER = 96
En arreglos se empieza desde 0?, temp[0]
Por medio de un ciclo es que podemos imprimir O leer cada dato en el arreglo
Al manejar los arreglos con una variable como N es mucho mas hacer cambios en el programa

Los arreglos se utilizan para almacenar datos como (ARREGLOS UNIDIMENSIONALES):
Listas de calificaciones, Inventarios de videojuegos, Lista de jugadores, Posiciones de un tablero, Datos de sensores

Las imagenes en computadoras se representan a partir de matrices, 0 representando el color negro, 255 el color blanco y 127 un tono gris, esto basandose en la escala de grises.
La estructura de matrices para imagenes a color es tridimensional

El arreglo bidimensional se representa como
int tablero[5][5];
